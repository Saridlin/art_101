Exercise 1

This is the first exercise completed for class where I experimented with using basic drawing functions such as rect(), triangle(), ellipse(); as well as, learning new functions: line() and point(). Each function is called when a certain condition is met, and it was a puzzle to figure out how to create 4 different states that the canvas would appear in: Triangle Idle, Ellipse Follow, Line Grid Follow, and Point & Rect Idle.
Each state was achieved through the use of conditions that checked the values of global variables such as mouseX, mouseY, and the canvas' width or height. The behavior of each state would depend on similar changing counters and the statistics of the generated "wallpaper" over time is tracked through console.log() prints.

Exercise 2

In this second exercise, I created an interactive game mimicking the classic "Snake" where the user controls a worm traveling across a grid to eat various fruits. I experimented with many different global variables that tracked the position of multiple fruits on the grid and whether the worm's head collided with them. If the worm eats, it prints some text that says "NOM!" and the fruit is randomly generated at a new location. If the worm goes too long without eating a certain fruit, it starts to complain to the user.
The game also tracks the user's key presses and mouse movements to execute different effects using global variable states such as keyIsPressed, mouseIsPressed and functions such as mouseMoved(). Through this, the game has extra cosmetic interactivity to make it more visually interesting. Changes in variable states and counters are also pushed to console.log().

Project 1

For Project 1, I worked off of the paint program template to create a program that allows the user to create and save their own works of art made in the browser.
There are 3 different algorithmic brushes available that can be used in various ways to achieve a psychedelic masterpiece. There is a brush that generates a continously rotating line around the position of the mouse, a rainbow-shifting triangular brush, and a spotted inverted brush that interacts with the colors already present on the canvas. With this, a surreal artpiece can be easily created by varying the size of the brush as well and creatively using the solid and translucent eraser tools.

The behavior of the brushes that I created were a very engaging challenge. I believe that I effectively made a fun program for users to play with and make art that interacts reflectively with them. The basics of a drawing program are there: making a mark, erasing marks, and recording your created piece.
What I found most challenging about this project was deciding upon the algorithmic behavior of my brushes and being able to execute what I had imagined them to be. I really had to explore the p5.js documentation to find what I needed and then experiment thoroughly with what I thought would work. I specifically learned how different transformative functions, like rotate(), need a translate() function to function with the coordinates I wanted the drawn shapes to be placed at. I also explored blendMode() and how DIFFERENCE and BLEND change the way drawn shapes affect the canvas well.
