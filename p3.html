Exercise 5

For this exercise, I worked with Kyra, Alisha and Minh to create a collection of data maps that we then used to create various images with different results.

I created a total of 8 custom functions to manipulate the grid arrays:
numMapRaw() - displays the array containing integers as a colored integer grid according to a given palette.
textMapRaw() - displays the array containing text data as colored strings.
numMaptoPalette() - displays the array containing integers as colored pixels and/or shapes according to a given palette.
textMaptoPallete() - displays the array containing text data as colored pixels and/or shapes.
monochromeMapNum() - displays the array containing integers as black and white pixels and/or shapes.
monochromeMapText() - displays the array containing text data as black and white pixels and/or shapes.
imageMapNum() - displays the array containing integers as images.
textMapNum() - displays the array containing text data as images.
I also created the functions checkText() and selectShape() to streamline some of the repeating code.

As a result, I have a total of 29 mapped images across the program. I used Kyra's Creeper gridArray, Alisha's Pig gridArray, and Minh's Fox gridArray. I had my own integer-based and text-based gridArrays for a total of 5 unique data arrays to use for these mapped images. These are all displayed across 4 unique pages that are accessible through the press of number keys 0-3.
This exercise was very tedious and repetitive, but it was a good assignment to learn how to manipulate large amounts of data contained within multidimensional arrays.

Project 3

For Project 3, I created a personal clock based on my tea-drinking habits and other patterns that I follow as I go about my day. There are small details that change while the hours and minutes pass such as:

  • Drinking Caffinated Tea during AM hours, and Decaf during PM hours
  • The Tea kettle begins to boil 4 minutes before the hour change and then fills the glass anew
  • The Reminders give a small insight into my personal schedule and important hours/periods in my day
  • The setting, a dining table by a window, is reminiscent of my favorite spot in my house: a breakfast nook with a view into a front yard with bushes and a tree where the change in the sky can be easily seen
The specific time visualizations are:
Dawn/Day/Sunset/Night - Change in Window Visual
Hours - Number of Tea Tags on Table
Minutes - Amount of Tea Consumed
Seconds - Ticking Tea Bag
Milliseconds - Rotating Sparkles on Metal Straw

I chose to draw almost all the assets for this project since depicting my glass, the scenery outside, and the tea kettle in vector graphics seemed a daunting task. However, this proved to be an obstacle and time sink, as well as causing my program to become laggy when open for long periods of time. The hour(), minute(), etc and map() functions were easy to understand and I did not have trouble dividing changes in my program using them. I found the most trouble in the creative visualization for this project and coming up with an idea for how to symbolically express each function of the clock.
